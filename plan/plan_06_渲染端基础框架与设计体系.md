## 计划概述

### 计划目标

## 技术方案

### 架构设计
页面路由层包裹全局布局；组件层复用 shadcn/ui；状态与数据请求在 hooks 层封装，动效策略集中管理。

### 核心技术选型
- React + Vite：快速构建与模块化
- Tailwind + shadcn/ui：一致的组件与样式
- Framer Motion：统一的进场与切换动效
- Zustand：轻量状态管理

### 数据模型
复用共享 schema，定义前端专用视图模型（如导航状态、主题状态、队列进度）。

### 接口设计
抽象 hooks：数据获取、错误提示、主题切换、导航控制；页面通过 hooks 访问 API 服务。
